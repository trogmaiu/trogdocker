É necessario ter o lamp instalado para executar este programa.
Ele  foi concebido sobre o Debian, sendo o docker.io. É necessário
mudar a configuração de permissão para poder executar, é uma
questão de segurança, que até diminui, mas ainda não encontrei
melhor alternativa. É necessário fazer um chmod +s docker.io para
poder executar. É necessário fazer uns tratamentos pois toda vez que roda
se cria no arquivo através do" fwrite".php várias procuras, ainda não criou-se
uma interface entre os dois mas já é possível fazer rodar sem nenhum tratamento.

Este exemplar, ou melhor à pasta 3, executei um filtro sobre o final do docker search,
resultando no arquivo resultado.txt, que resultou no arquivo fazer.sh que foi trabalhado
para ter o final em variável para ser posto em outro arquivo .php que vai ser trabalhado, 
possivelmente para originar uma tabela <th> <td>.
